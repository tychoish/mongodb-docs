=========
rs.conf()
=========

.. default-domain:: mongodb

.. method:: rs.conf()

   :returns: a :term:`document` that contains the current
             :term:`replica set` configuration document.

   Wraps :dbcommand:`replSetGetConfig`. See :doc:`/reference/replica-configuration`
   for more information on the replica set configuration document.

.. method:: rs.config()

   :method:`rs.config()` is an alias of :method:`rs.conf()`.
